Market Data Analysis & Recommender

This is a Market Basket Analysis and Product Recommendation tool built in Python. Kivi Game (Academic Project)

This is a Java Swing Kivi board game developed as an academic course project for educational and demonstration purposes only. It features a multi screen interface, and multiple color themed UI, with implemented game logic, scoring, and dice combination validation for both human and AI players. The game includes save/load functionality, a winner screen, and is packaged as a runnable JAR. It was built in a team of five students, along with full software documentation as part of the course requirements.

Weather Data Analysis

This is a Python application for importing and analyzing historical weather data from the Meteostat library. It uses SQLite with a custom database schema to store data. The tool provides analyses like monthly summaries, temperature extremes, rainy day counts, and cross city comparisons, and offers both a command line interface and a Tkinter GUI to add cities, import data, and run climate analyses.

Tech: Python, Meteostat, SQLite
